# zsh-tmux-rename

A zsh plugin to rename tmux window automatically.

## How it works

[![asciicast](https://asciinema.org/a/146359.png)](https://asciinema.org/a/146359)

## Installation

### Example using zplug

Add the following command to your `.zshrc`.

```zsh
zplug 'sei40kr/zsh-tmux-rename'
```
